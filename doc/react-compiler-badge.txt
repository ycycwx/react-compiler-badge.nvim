INTRODUCTION                                  *react-compiler-badge*

This plugin highlights React components that have been optimized by the
React Compiler (experimental). It uses `babel-plugin-react-compiler` to
detect optimized functions.

SETUP                                         *react-compiler-badge-setup*

If you are using lazy.nvim:
>
    {
      "ycycwx/react-compiler-badge.nvim",
      build = "npm ci",
      opts = {},
    }
<

CONFIGURATION                                 *react-compiler-badge-config*

You can configure the highlights and icon by passing a table to the setup
function:
>
    require("react-compiler-badge").setup({
      highlight = {
        bg = "#1a6bbc",
        fg = "#ffffff",
        bold = true,
      },
      icon = " Memo✨ ",
      failed_highlight = {
        bg = "#8a1f11",
        fg = "#ffffff",
        bold = true,
      },
      failed_icon = " Memo✕ ",
      show_failed = true,
      show_diagnostics = true,
    })
<

Options:

highlight~
  Highlight group definition for optimized components.

icon~
  Virtual text icon for optimized components.

failed_highlight~
  Highlight group definition for component-like functions that were not
  optimized by React Compiler.

failed_icon~
  Virtual text icon for component-like functions that were not optimized by
  React Compiler.

show_failed~
  Whether to show failed optimization indicators. Defaults to true.

show_diagnostics~
  Whether to publish React Compiler failure reasons as Neovim diagnostics.
  Defaults to true.
